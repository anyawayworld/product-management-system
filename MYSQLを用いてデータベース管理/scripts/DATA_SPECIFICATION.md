## 🎛️ カスタマイズ方法

取引件数を変更したい場合：

\`\`\`sql
-- 04_generate_inventory_transactions.sql の以下の行を変更
WHILE i < 15 DO -- ← この数値を変更
-- 例: 5件にする場合は WHILE i < 5 DO
\`\`\`

期間を変更したい場合：

\`\`\`sql
-- 以下の行を変更
SET random_date = DATE_SUB(CURDATE(), INTERVAL FLOOR(RAND() * 365) DAY);
-- 例: 6ヶ月にする場合は * 180 DAY
